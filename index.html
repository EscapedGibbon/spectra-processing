src/x/xNoiseSanPlot.js

Determine noise level by san plot methodology (https://doi.org/10.1002/mrc.4882)

xNoiseSanPlot(data: Array, options: object)
Parameters
data (Array) real or magnitude spectra data.
options (object = {})
Name Description
options.mask array? boolean array to filter data, if the i-th element is true then the i-th element of the distribution will be ignored.
options.scaleFactor number (default 1) factor to scale the data input [i] *=scaleFactor.
options.cutOff number? percent of positive signal distribution where the noise level will be determined, if it is not defined the program calculate it.
options.factorStd number (default 5) factor times std to determine what will be marked as signals.
options.refine boolean (default true) if true the noise level will be recalculated get out the signals using factorStd.
options.fixOffset boolean (default true) If the baseline is correct, the midpoint of distribution should be zero. if true, the distribution will be centered.

xyAlign

src/xy/xyAlign.js

xyAlign will align data of two spectra by verifying wether x values are in a certain range (delta). The two spectra should not have two consecutive x values which difference is smaller than delta to achieve good results!

xyAlign(spectrum1: DataXY, spectrum2: DataXY, options: object)
Parameters
spectrum1 (DataXY) First spectrum data
spectrum2 (DataXY) Second spectrum data
options (object = {})
Name Description
options.delta number (default 1) The range in which the two x values of the spectra must be to be placed on the same line
options.common boolean (default true) If true , only the data considered as common to both spectra is kept. If false , the data y arrays are completed with zeroes where no common values are found
options.x string (default 'x1') Defines what x values should be kept ( x1 : spectrum 1 x values, x2 spectrum 2 x values, weighted : weighted average of both spectra x values)
options.weightFunction function (default undefined) Function that allows to weight delta depending on the X values of the spectrum

xyCalibrate

src/xy/xyCalibrate.js

Calibrates the data based on a range and means of peaks in this range Based on a range we will make a peak picking using global spectra deconvolution The selected nbPeaks will then be taken into account to calculate an average X value. The difference between the targetX and the averageX value will be returned

xyCalibrate(data: DataXY?, range: Object, options: Object): number
Parameters
data (DataXY?) array of points {x,y}
range (Object = {})
Name Description
range.from number? Beginning of the range where the interest signal is localed
range.to number? End of the range where the interest signal is localed
options (Object = {})
Name Description
options.nbPeaks number (default 1) Number of peaks to consider to calculate mean (sorted by height)
options.targetX number (default 0) Expected value for the mean of the peaks position
options.gsd number (default {}) GSD options. You may check options here: http://mljs.github.io/global-spectral-deconvolution/#gsd
options.gsd.minMaxRatio number (default 0.2) GSD Threshold to determine if a given peak should be considered as a noise.
Returns
number: difference between targetX and currentX
